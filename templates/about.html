Linear Regression

This is a linear regression system. It is using machine learning algorithm in the backend. I am using Sklearn libraries to do my work easily. The language that I have used in this project is python.

  1. LinearRegression : This library is used to plot a linear regression graph between two axis in a dataset. By that linear regression graph, we can easily find the value of any depended variable by knowing the independent variable
  2. train_test_split : The job of this library is to split the data of dataset intp train and test. the larger portion of the data is used to train the dataset and the remaining part is used to test the dataset.
By using these methods we can plot the linear regression graph and by using that graph we can find the value of any unknown variable. In this project, I am using a dataset of home price.
